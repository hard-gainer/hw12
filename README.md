# Задание

Доработайте ваш проект FastAPI для работы с БД. Для выполнения фоновых задач используйте BackgroundTasks или Celary.

- Шаг 1. При работе с SQLAlchemy вы создавали скрипт для наполнения БД данными из csv файла. Теперь создайте эндпойнт в проекте FastAPI выполняющий эту процедуру как фоновую задачу. В качестве параметров он должен принимать путь к csv файлу

- Шаг 2. Так же добавьте эндпойнт позволяющий удалить записи из БД по переданному списку. Выполнение процедуры удаления также следует выполнять как фоновую задачу.

- Шаг 3. Подключите к вашему проекту СУБДД redis и добавьте процедуру кеширования для всех эндпойнтов возвращающих данные пользователю.
